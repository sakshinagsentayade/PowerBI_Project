# Description
# Customer Churn Analysis Dashboard – Power BI

This project presents an interactive Customer Churn Analysis Dashboard built using Power BI to identify customer retention patterns and uncover factors contributing to churn. The dataset was imported from CSV files and processed using Power Query for data cleaning, transformation, and handling missing values.

Key performance indicators (KPIs) and DAX measures were created to analyze churn rates, customer demographics, and business trends. The dashboard features interactive visualizations, including KPI cards, bar and column charts, pie and donut charts, heatmaps, slicers, and filters, enabling users to explore data dynamically.

The analysis provides valuable insights into customer behavior, demographic segments, and churn trends, helping businesses make informed decisions and develop effective customer retention strategies. This project demonstrates skills in data preprocessing, data modeling, DAX calculations, dashboard design, and business intelligence reporting using Power BI.
